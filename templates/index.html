⑤ How It Works — Technical Breakdown
1

LangGraph State Machine

Two nodes connected in a directed acyclic graph using StateGraph(AgentState). The AgentState TypedDict object flows through both nodes, accumulating data at each step.

2

Pydantic Output Guardrail

PydanticOutputParser(AlertOutput) is injected into the LangChain chain as prompt | llm | parser. If the LLM outputs anything outside the schema, the library raises OutputParserException — no raw text ever escapes.

3

Multi-Model Waterfall

The pipeline maintains a priority list of 5 free OpenRouter models. If the top model is rate-limited (HTTP 429), it automatically skips to the next — guaranteeing a result without paid API credits.

4

CVSS Threshold Rule

The prompt explicitly instructs the LLM: "Set is_critical = true if CVSS score > 8.0". With a score of 9.8, the expected output is always true, validated post-response.
